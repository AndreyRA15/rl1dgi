Разработка сервера и клиента для отображения данных из файла wav по TCP

Сервер должен иметь возможность по запросу клиента в виде SCPI команд выполнять следующие действия:
1. Загружать wav файл по имени полученному от клиента
2. Возвращать по запросу клиента частоту дискретизации и количество отчетов
3. По запросу клиента возвращать все отчеты сигнала из файла
4. Обрабатывать элементарные ошибки (нет такой команды, нет такого файла, и прочее)

Клиент должен функционировать следующим образом:
1. Подключиться к серверу по указанному порту
2. Отправить запрос на загрузку файла по имени
3. Запросить частоту дискретизации и количество отчетов
4. Запросить отчеты из файла и построить спектрограмму по полученным данным.
